# Project Remo Web v2

Consumer and investor pages for Project Remo v2.

- Consumer page: `/`
- Investor page: `/for-investors/`
- Deployment: GitHub Pages via the workflow in `.github/workflows/deploy-pages.yml`

All images and videos are served from this repository with relative paths. The site has no third-party font or media dependency, so the core pages remain usable on networks where Google Fonts or public CDNs are unavailable.

## Updating

Make website changes in this repository, then commit and push to `main`. GitHub Pages redeploys automatically after each successful workflow run.
